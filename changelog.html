Changelog

  • Version 3.0
    • Cleaned up GUI and made it compatible with all operating systems
    • Removed Store url param
    • Removed MySQL database to log server lists
    • Now shows player heads with usernames when you hover over them
    • Modified layout, can now be expanded vertically for more chat room
    • Changed default size from 415px to 420px
    • Fixed bugs (because bugs need to be fixed!)
    • Made the yellow color darker and thus more readable
  • Version 2.0
    • Neater, smaller GUI
    • Better connecting to server system
    • Temporarily storing join/leave information so it can join other servers quicker
    • Support for formatted text (bold, italics etc.)
    • Fixed up typos
    • List of servers which can be defined by the person who embedded WMcChat into the webpage
  • Version 1.2
    • 'Initial release'
