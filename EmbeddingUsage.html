Embedding manual

Embedding help goes here - code goes here - param info goes here

To embed this into your website, you will need this 'template' code below:

<embed id="WMcChat" 
type="application/x-java-applet;version=1.6" 
width="415" 
height="600" 
archive="https://dl.dropboxusercontent.com/u/77591131/WMcChat/WMcChat.jar,https://dl.dropboxusercontent.com/u/77591131/WMcChat/mcprotocollib-1.7.4-SNAPSHOT.jar" 
code="com.droppages.Skepter.Start" 
pluginspage="http://java.com/download/"> 
<param name="permissions" value="all-permissions"> 
<param name="servers" value="server:port,server:port">
<param name="storeurl" value="true">

Looks simple? Actually, there's a bit more work you need before it can work to your full taste. To use this, you need to adjust the param values:

  • Permissions
    • This line of code is required in order for it to work - nothing needs to be adjusted here
      • <param name="permissions" value="all-permissions">
  • Servers
    • WMcChat allows you to add a list of servers to the application, so they can only join that list. To adjust this value, you need to enter the server IP, followed by a (:) and then the port (default it to 25565 if you don't know your port). Then, you can add another server by separating it with a comma (,). Please don't use spaces!
