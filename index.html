WMcChat

Hello there, welcome to my website for WMcChat! Standing for Web Minecraft Chat, WMcChat is a Minecraft chat client, similar to MineChat, except for the web!

Why use WMcChat?

We have Minecraft. We have MineChat. Why make another chat client? WMcChat aims to be different. Unlike the other clients:

  • WMcChat does not store any files on your computer (unlike Minecraft)
  • WMcChat does not require downloading anything (unlike Minecraft)
  • WMcChat is not resource intensive (unlike Minecraft)
  • WMcChat can be put on a website - see usage below
  • WMcChat can be used on computers (unlike MineChat)

The 'param name="servers"' part is special. It allows you to put a list of your own servers in there so people can't access other servers on your webpage. To use, replace the 'server:port' with your own server IP and port (put it as 25565 if you don't know). You can add as many servers are you want, as long as they are separated by a comma (,) and DON'T contain any spaces!
