WMcChat

Hello there, welcome to my website for WMcChat! Standing for Web Minecraft Chat, WMcChat is a Minecraft chat client, similar to MineChat, except for the web!

Why use WMcChat?

We have Minecraft. We have MineChat. Why make another chat client? WMcChat aims to be different. Unlike the other clients:

  • WMcChat does not store any files on your computer (unlike Minecraft)
  • WMcChat does not require downloading anything (unlike Minecraft)
  • WMcChat is not resource intensive (unlike Minecraft)
  • WMcChat can be put on a website - see usage below
  • WMcChat can be used on computers (unlike MineChat)
  • WMcChat is free and always will be forever! (You need a Minecraft account first though) (donations will be appreciated, except I can't take donations at this stage) (unlike MineChat's pro features)
  • WMcChat is customisable - you can add servers to it when it is embedded in a website so it is more personal to the server

FAEAQ (Frequent and easily answered stuff)

  • Problem: It's a scam! They store your password and try to hack your account!
    • Answer: No, seriously guys - this is not a scam. It uses the same code as the regular Minecraft login and is 100% safe. I'm a good person and good people do not steal Minecraft accounts.
  • Problem: It says Error - see for details, then it says something about Major Minor 51.0 or something like that
    • Answer: You are using Java 6. You need Java 7 to use this. - You can download it here
  • Problem: Where's the download link?
    • Answer: This isn't a program, it's a web applet. You have to embed it into a website.
  • Problem: When I get kicked from a server it sometimes says something like \U0078
    • Answer: I'm working on it, except fixing it would require adding a new library which I'm not ready for at this stage.
  • Problem: I found a bug - something isn't working correctly
    • Answer: Congratulations! Tell me about it - click Contact owner at the top and send him an email with the problem.
